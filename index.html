Dokumentační web, který prostě funguje.

Tahle verze je postavená jako čistý statický web. Po pushi do repozitáře se automaticky nasadí na GitHub Pages, bez build chyb a bez závislosti na problematickém runtime nastavení.

Co je v balíčku

Stránka je navržená jako lehký dokumentační základ s čistým layoutem, responzivitou a jednoduchou údržbou.

1

Moderní vzhled

Gradienty, cards, přepínání motivu a přehledná typografie bez těžké knihovny.

2

Jednoduché nasazení

Workflow posílá obsah přímo do větve gh-pages po každé změně v hlavní větvi.

3

Bez build pádu

Odpadá problém s Nuxtem a nesprávným načítáním modulů při generování statické verze.

Deploy v praxi

Stačí pushnout změny do větve main. GitHub Actions vezme složku site/ a publikuje ji na Pages.

Krok 1

Upravit obsah

Texty, menu i vzhled jsou v běžných HTML/CSS souborech.

Krok 2

Pushnout do main

Workflow se spustí samo a začne publikovat novou verzi.

Krok 3

Publikovat na Pages

Výsledek běží z gh-pages branch a funguje i jako projektová stránka.

git add . && git commit -m "update site" && git push
