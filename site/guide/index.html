Průvodce

Jak web nasadit a udržet

Tohle je nejjednodušší varianta: upravíš HTML/CSS, pushneš na GitHub a workflow publikuje novou verzi.

Obsah

Uprav texty

Domovská stránka i průvodce jsou běžné HTML soubory. Bez builderu a bez skriptů navíc.

Styly

Uprav vzhled

Všechny barvy, radiusy i layout jsou v jednom souboru assets/style.css.

Deploy

Pošli změny

GitHub Actions automaticky zveřejní adresu z branch gh-pages.

Doporučený postup
  1. Změň obsah v site/index.html a site/guide/index.html.
  2. Podle potřeby uprav site/assets/style.css.
  3. Commitni a pushni do main.
  4. Počkej na dokončení workflow a otevři GitHub Pages URL.
