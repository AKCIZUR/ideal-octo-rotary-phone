# První kroky

Tahle sekce je připravená jako ukázka struktury dokumentace.

## Jak pokračovat

1. Nahraď obsah vlastními stránkami v `content/`.
2. Uprav nadpisy a navigaci podle projektu.
3. Po pushi do `main` se vše automaticky přepublikuje na GitHub Pages.

Více informací o Docus najdeš na [docus.dev](https://docus.dev).
