# FAQ

## Je to připravené pro GitHub Pages?

Ano. Workflow pro GitHub Actions je součástí projektu.

## Musím psát HTML?

Ne. Obsah je připravený pro Markdown a MDX.

## Dá se to přizpůsobit jinému projektu?

Ano. Stačí změnit název, logo, odkazy, sidebar a barevné proměnné.

## Lze přidat další sekce?

Ano. Přidej nový soubor do `src/content/docs/` a uprav sidebar.

## Je šablona vhodná jako dlouhodobý základ?

Ano. Je postavená jako jednoduchá statická dokumentace s minimální údržbou.
