# Konfigurace

## `astro.config.mjs`

Tady je hlavní konfigurace Starlightu:

- název webu
- popis
- logo
- favicon
- sidebar
- sociální odkazy
- vlastní CSS
- edit link

## `src/content.config.ts`

Tento soubor zapíná dokumentační collection pro Starlight:

- `docsLoader()`
- `docsSchema()`

## `src/styles/custom.css`

Sem patří vlastní vzhled:

- tmavé pozadí
- barevné akcenty
- úpravy šířky obsahu
- drobné vizuální detaily

## `BASE_PATH` a `SITE_URL`

Tyto proměnné se používají při buildu na GitHub Pages:

- `SITE_URL` = veřejná adresa webu
- `BASE_PATH` = podadresář repozitáře

Pro uživatelské Pages může být `BASE_PATH` prázdné.
