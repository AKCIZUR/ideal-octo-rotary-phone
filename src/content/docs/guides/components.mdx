## Připravené komponenty

- `Hero.astro`
- `FeatureGrid.astro`

## Jak je používat

Komponenty jsou navržené pro MDX úvodní stránku, takže je můžeš skládat jako stavebnici:

```mdx
import Hero from '../../components/Hero.astro';
import FeatureGrid from '../../components/FeatureGrid.astro';
```

## Z čeho jsou složené

- Astro komponenty
- Lucide SVG ikony
- čisté CSS bez heavy UI frameworku

## K čemu se hodí

- landing page
- announce sekce
- feature přehled
- docs home page
