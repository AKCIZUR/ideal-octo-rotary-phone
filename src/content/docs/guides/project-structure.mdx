# Struktura projektu

Doporučený základ:

```text
src/content/docs/
├── index.mdx
├── getting-started.mdx
├── configuration.mdx
├── deployment.mdx
├── faq.mdx
├── changelog.mdx
└── guides/
    └── project-structure.mdx
```

## Proč takto

- jednoduché hledání souborů
- přehledný sidebar
- snadné rozšiřování
- dobré pro GitHub-based workflow

## Jak škálovat dál

Můžeš přidat například:

- `api/`
- `tutorials/`
- `reference/`
- `examples/`

Každá složka může být samostatná sekce ve sidebaru.
