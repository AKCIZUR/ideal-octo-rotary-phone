## Jak se počítá URL

Astro konfigurace si čte GitHub Actions prostředí a odvozuje:

- `site` z `GITHUB_REPOSITORY` a `GITHUB_SERVER_URL`
- `base` z názvu repozitáře
- `editLink` z branch a repo URL

Pokud je repo typu `owner.github.io`, použije se root site bez repo base path.

## Workflow

Workflow používá:

- `actions/checkout`
- `withastro/action`
- `actions/deploy-pages`

## Co nastavit v GitHub repozitáři

V nastavení Pages vyber:

- **Source**: GitHub Actions

To je celé. Žádný ruční build output branch a žádné ruční přepisování odkazů.

## Lokální kontrola před push

```bash
pnpm build
pnpm preview
```
