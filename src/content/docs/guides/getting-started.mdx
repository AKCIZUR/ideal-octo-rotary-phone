## Instalace

```bash
pnpm install
```

## Lokální vývoj

```bash
pnpm dev
```

## Co by mělo fungovat hned po klonu

- základní navigace v levém sidebaru
- dark theme bez další konfigurace
- Lucide ikony na úvodní stránce
- build přes `pnpm build`
- deploy přes GitHub Actions

## Kontrolní seznam

- Node.js 22.12+ nebo 24
- pnpm
- GitHub Pages zapnuté v repozitáři
- `main` branch jako deployment branch
