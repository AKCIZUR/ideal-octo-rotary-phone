## Co dělá tmavý vzhled

Dark theme je postavený na:

- CSS proměnných
- lehkém skleněném pozadí
- fialovo-azurovém akcentu
- zaoblených kartách
- jemných hranách a stínech

## Kde upravit vzhled

- `src/styles/custom.css`
- `src/assets/vccsd-mark.svg`

## Doporučený postup

1. uprav barvy v CSS proměnných
2. dolaď logo
3. změň hero texty
4. přidej nové stránky do sidebaru

## Praktický důsledek

Šablona zůstane vizuálně konzistentní i při přidávání dalších sekcí nebo referenčních stránek.
