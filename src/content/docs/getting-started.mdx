# Začínáme

Tato šablona je připravená jako výchozí dokumentační web pro nové i existující projekty.

## Spuštění lokálně

```bash
npm install
npm run dev
```

## Struktura obsahu

Starlight načítá dokumentaci ze složky:

```text
src/content/docs/
```

Každý `.md` nebo `.mdx` soubor se změní na samostatnou stránku.

## Co upravit jako první

- `astro.config.mjs`
- `.github/workflows/deploy.yml`
- `src/content/docs/index.mdx`
- `src/styles/custom.css`

## Tip

Když chceš další sekce, stačí vytvořit nové soubory nebo podsložky v `src/content/docs/` a přidat je do sidebaru.
