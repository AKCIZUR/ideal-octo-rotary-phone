## Rychlý start

```bash
pnpm install
pnpm dev
```

## Co se děje při deployi

1. GitHub Actions spustí build.
2. Astro si přečte `GITHUB_REPOSITORY` a dopočítá `site` i `base`.
3. Site se vygeneruje jako statická dokumentace.
4. GitHub Pages dostane hotový artifact a publikuje ho.

## Doporučený provozní režim

- vývoj lokálně přes `pnpm dev`
- build kontrola přes `pnpm build`
- před nasazením ověřit `pnpm preview`

> Tohle je užitečné jako základ pro docs, interní knowledge base i veřejný open source projekt.
