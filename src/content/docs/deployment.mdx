# Nasazení

## Jak to funguje

1. push do `main`
2. GitHub Actions spustí build
3. Astro vytvoří statický výstup
4. GitHub Pages nasadí web

## Workflow

Soubor `.github/workflows/deploy.yml` už je připravený.

### Co je potřeba změnit

- `USERNAME`
- `REPO`

## Doporučený postup

Po nahrání repozitáře na GitHub:

1. otevři **Settings**
2. přejdi na **Pages**
3. nastav **GitHub Actions** jako zdroj
4. proveď první push do `main`

## Poznámka

Pokud používáš projektové Pages, musí odpovídat i `BASE_PATH`.
