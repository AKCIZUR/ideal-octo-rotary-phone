---
title: Repository detection
description: Technická reference pro automatickou detekci repository a URL.
---

`astro.config.mjs` používá tyto environment proměnné:

- `GITHUB_REPOSITORY`
- `GITHUB_SERVER_URL`
- `GITHUB_REF_NAME`
- `SITE_URL`
- `ASTRO_BASE`

Když je repo veřejně hostované přes GitHub Pages, base path se odvodí z názvu repozitáře. U user pages se base path vypne.
